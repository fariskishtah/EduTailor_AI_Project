# EduTailor AI Project

This project predicts student performance using Machine Learning and generates personalized learning recommendations for the EduTailor platform.

## Dataset
Student Performance Dataset

## Features Used
- Study Time
- Failures
- Absences

## Model
Linear Regression

## Outputs
- Predict final grade
- Classify student performance
- Generate learning recommendations

## Technologies
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
